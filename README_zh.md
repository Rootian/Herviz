Herviz 是一个北美租车网站，提供租车服务、订单管理服务以及用户中心服务

**中文** | [English](https://github.com/Rootian/Herviz)



## 技术栈

|   模块   |           框架            |
| :------: | :-----------------------: |
| 前端框架 |         Bootstrap         |
| 后端框架 | Springboot + Mybatis plus |
| 权限框架 |         Sa-token          |
| 缓存框架 |           Redis           |
|  数据库  |           Mysql           |
